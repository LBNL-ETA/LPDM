Running Simulations

Configuration
-------------

run_time_days
_____________
The number of days to run the simulation for.

.. csv-table::
   :header: "Data Type", "Range", "Units", "Default Value"
   :widths: 40, 40, 40, 40

   int, n/a, days, 7 days
