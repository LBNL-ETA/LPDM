  config,

  override_args

Reads in all the information from the JSON about all listed EUD's and creates their connections.

Since each EUD might have a different construction signature, this function references the "Eud Dictionary" which has a list of the names of different unique construction parameters for each EUD, and then tries to find the values for those parameters in the configuration file

Parameters
configthe configuration input dictionary derived from the JSON parameter file
runtimethe duration of the simulation (in seconds)
override_argsthe dictionary of override arguments
Returns
list of tuples of EUD_id's, list of that device's connected devices.
